# Web Security Learning Repository

Hi, I'm a cybersecurity student documenting my journey in web security and bug bounty hunting.

This repository contains my notes, lab write-ups (generalized), and practical examples based on platforms like PortSwigger Web Security Academy.

## Topics Covered

- File Upload Vulnerabilities
- Authentication and Access Control
- Injection Attacks
- Cross-Site Scripting (XSS)
- Security Misconfigurations

## Tools Used

- Burp Suite
- Browser DevTools
- Custom payloads

## Goals

- Understand real-world web vulnerabilities
- Practice ethical hacking techniques
- Build a strong foundation for bug bounty hunting

## Disclaimer

This repository is for educational purposes only.
All techniques are tested in legal lab environments only.
